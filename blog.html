Discuss the scope of var, let, and const

Ans:- While let and const are block-scoped, var declarations are either globally scoped or function-scoped. Let variables can be updated but not re-declared, const variables cannot be updated or re-declared, and var variables may both be updated and re-declared inside their scope.

Tell us the use cases of null and undefined

Ans:- Conclusion. Null and undefined are two special values in JavaScript that represent the absence of a value or object. The key difference between the two is that null is a value that can be assigned to a variable, while undefined is the default value assigned to a variable when no value is explicitly assigned to it.
